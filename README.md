## Project setup
```
composer install
```

### Create enviroment file
```
cp .env.example .env
```

### Generate Unique Application Key
```
php artisan key:generate
```

### Run PHP built-in
```
php artisan serve
```

See [Configuration Reference](https://laravel-news.com/creating-configuration-in-laravel).
